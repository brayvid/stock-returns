Formatting Rules

Single Assets

Enter comma-separated tickers.
Example: AAPL, BTC-USD, GC=F

Equal-Weighted Portfolios (Quarterly Rebalance)

Wrap tickers in parentheses to create an equal-weighted portfolio.
Example: (AMZN, AAPL, GOOGL)

Custom Weighted Portfolios (Quarterly Rebalance)

Use multipliers like 0.6*TICKER inside parentheses. If weights don't add up to 1.0, any unweighted tickers will evenly split the remainder.

Fully-weighted example: (0.6*SPY, 0.4*TLT)
Partially-weighted example: (0.5*VTI, VOO, QQQ)

High Yield Savings Account Simulation

To plot a baseline savings yield with daily compounding, check the Include High Yield Savings checkbox and set your target annual yield directly on the control panel.
